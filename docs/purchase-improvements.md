## Решения по дизайну

- **Курс:** показываем оба источника, а не один «лучший». Перекупщик видит дельту над банковским — это для него прозрачность.
- **Дубль IMEI:** не блокируем, только warning. Бэк всё равно валидирует на 409 при `in_stock`. Перекупу полезно знать что аппарат к нему уже возвращался.
- **Черновик:** только один, не история. Если хочется истории — отдельная фича позже.
