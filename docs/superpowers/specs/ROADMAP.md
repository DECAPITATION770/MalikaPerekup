## 2. Вкладка «Telegram-пользователи» + блокировка ✅ РЕАЛИЗОВАНА

**✅ Шаг 1 — блокировка пользователя** (2026-06-13, merge `c3121ba`).
`users.is_blocked`/`blocked_at`, JWT-клейм `src`, гард в `get_current_user_id`
(telegram → 403 `user_blocked`, немедленно), отказ на `/auth/telegram`, admin
block/unblock. Логин/пароль не затрагивается. Спец/план
`{specs,plans}/2026-06-13-user-blocking-*`.

**✅ Шаг 2 — вкладка «Telegram-пользователи»** (2026-06-13, merge `8cd1995`).
Расширили страницу «Пользователи»: реальные аватары (бот тянет при входе →
MinIO → presigned), кликабельные `t.me/<username>`, подробный статус-бейдж
«клиент» (`client_status`: no_shop/trial/client/expired/frozen). Спец/план
`{specs,plans}/2026-06-13-tg-users-tab-*`.

**✅ Шаг 3 — редактируемые контакты владельца** (2026-06-13, merge `da30fc8`).
`users.admin_contact_note` (миграция `0021`), `PATCH /admin/users/{id}/contact`,
кнопка «Контакты» на ShopDetail (редактирует телефон + заметку), заметка видна
на Users. Работает для не-Telegram тенантов. Спец/план
`{specs,plans}/2026-06-13-owner-contact-*`.

**Сознательно отложено (не в scope):** сбор телефонов через `requestContact`
(телефон вводит/правит админ вручную). Если понадобится — отдельный цикл.
